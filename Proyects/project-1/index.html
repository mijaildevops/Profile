Desplegar una VM con ubuntu En UpCloud

About This Project

En este proyecto enseño como crear y desplegar una maquina virtual con ubuntu en UpCloud (compa#ina de hosting de servidores en la nube), Adema de como conectarse por SSH con User and Password.

Los recursos en la nube ofrecen grandes ventajas sobre las opciones tradicionales de servidores compartidos o dedicados. Estos recursos se pueden ampliar o reducir en consecuencia (escalado), lo que lo hace más flexible y, por lo tanto, más rentable

Steps

  • 1. Crear cuenta en upcloud
  • 2. Login upcloud
  • 3. Click en tab Servers
  • 4. Click en el botón Deploy-Server
  • 5. Select Location (Cualquier región)
  • 6. Select Plan and Storage
  • 7. Select Operating system
  • Ubuntu 20.04 LTS (Focal Fossa)
  • 8. Click en more settings, si quieres cambiar en método de login
  • 9. Ingresa Hostname y descripción del servidor. (debe ser minúscula todo)
  • 10. Click en deploy server
  • 11. Abrir Putty (Windows)
  • 12. Ingresar la IP de la Maquina Virtual Creada
  • Verificar el puerto 22
  • 13. Ingresar User y Pass (UPCLOUD envía los datos de acceso por email)
  • ifconfig
  • 14. Verificar Distribución de linux instalada
  • lsb_release -idc
  • 15. Verificar IP Pública
  • curl ifconfig.me
  • 16. verificar Ubicación según la IP
  • curl https://ipvigilante.com/la-dirección-IP-VM
  • 17. Ping de red
  • ping mijailosorio.com
  • 18. Ver la información detallada respecto al dominio consultado
  • apt install whois
  • whois mijailosorio.com
