SSH Server

About This Project

En este proyecto enseño como crear un servidor SSH con ubuntu. SSH es un protocolo seguro para la administración remota de servidores a través de la consola o terminal.

El protocolo SSH proporciona confidencialidad (los datos van cifrados punto a punto), autenticación (podremos autenticarnos frente al servidor SSH de múltiples maneras, con usuario/clave, criptografía de clave pública e incluso podremos configurar un segundo factor de autenticación)

Steps

  • 1. El servidor está en una máquina Virtual con Ubuntu
  • 2. Para virtualizar utilizaremos VirtualBox
  • 3. Iniciar VM
  • 4. Abrir Terminal (Ctrl + Alt + T)
  • 5. Iniciar sesión como root
  • Sudo Su
  • 6. Actualizar paquetes ubuntu
  • sudo apt update
  • 7. instalar openssh-server
  • sudo apt install openssh-server
  • 8. Verificar Status Instalación
  • sudo systemctl status ssh
  • 9. Cambiar Puerto de Conexión (Por seguridad)
  • nano /etc/ssh/sshd_config
  • configurar 2222 (puede ser cualquiera)
  • 10. Reiniciar el Servicio SSH
  • /etc/init.d/ssh restart
  • 11. Habilitar Firewall
  • sudo ufw enable
  • 12. Abrir puerto SSH Firewall
  • sudo ufw allow 2222
  • 13. Verificar IP del servidor
  • ifconfig
  • 14. Abrir Putty en “Windows”
  • Ingresar la IP del servidor SSH
  • Cambiar puerto de conexión
  • Click en el botón “Open”
  • 15. Crear carpeta desde línea de comando (consola de putty)
  • mkdir SSH-Folde
  • 16. Descargar imagen desde línea de comando (consola de putty)
  • wget https://www.diplomadosonline.com/wp-content/uploads/2020/02/What_is_Data_Science-1080x675.jpg
